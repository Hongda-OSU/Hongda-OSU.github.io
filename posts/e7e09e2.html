什么是量化

  • 量化交易 = 用数据+统计+程序，系统化地找出历史上大概率盈利的规律，然后自动执行。核心是去除人为情绪，靠模型和算法驱动决策。起源于美国，国内近年快速兴起。

什么是量化交易

  • 核心问题：市场 = 负和游戏（手续费是house edge），随机交易必亏。
  • 解法：找到 win_rate > 50% 的信号，只在有优势时入场。
  • 量化交易本质：从历史数据里挖出 P(win) > 0.5 的规律，系统化执行，去掉人为情绪干扰。
  • 核心思想：模型会过期，市场会变，策略必须持续迭代。
  • 三个关键模块
    • 策略回测 → 验证历史胜率
    • 仓位管理 → 控制每次下注大小
    • 止盈止损 → 保护期望收益

不是赌博，是在找一个有正期望值的游戏，然后反复执行它。

量化交易基本流程

数据采集 → 数据清洗 → 量化模型 → 交易信号 → 执行下单

  • 三个核心环节

    • 数据层 — 历史行情 + 基本面 + 新闻资讯，清洗后喂给模型
    • 模型层 — 数学建模 + 策略编程，分析数据产生信号
    • 执行层 — 信号转化为具体指令：买什么、何时买、买多少、何时卖

  • 对程序员来说本质就是一个 pipeline：数据进，信号出，自动执行。

回测的四大陷阱

  • 交易者效应 — 你的”漏洞”被其他人发现后就被修补了。
  • 随机效应 — 过拟合了噪声，不是真规律。
  • 最优化矛盾 — 参数调出来的是历史最优，不是未来最优。
  • 过度拟合 — 训练集准确率100%，测试集崩了。

回测只是参考，不是答案。量化的真正壁垒是持续学习、持续更新模型的能力，而不是找到一个”永动机策略”。

量化交易发展历程

  • 索普（Edward Thorp） — 开创者。数学家出身，先用概率+凯利公式横扫拉斯维加斯，1967年出版量化交易开山之作，1969年创立史上第一家量化对冲基金。本质是把赌场的数学优势逻辑搬到了华尔街。

  • 西蒙斯（James Simons） — 集大成者。1988年创立文艺复兴科技，旗舰大奖章基金1989-2009年均回报约35%，远超巴菲特、索罗斯，2008年金融危机照样稳。把量化交易推向了这个时代的主流。

  • 索普开创 → 量化产品涌现（指数基金、对冲基金、FOF）→ 西蒙斯验证并推广 → 成为全球资管主流

量化不是万能的，西蒙斯自己的基金也有亏损年份。优势在于长期概率，不在于每次必赢。

随机漫步理论

  • 核心思想：不可预测 ≠ 不可量化
  • 想象一个醉汉站在街道中间，每走一步完全随机：往前或往后概率各50%。

对应股价

  • 每天涨跌 = 醉汉每步前进后退
  • 今天股价 = 醉汉当前位置

随机漫步理论说股价每步随机，无法预测具体走向。但索普发现，虽然不知道醉汉走到哪，但知道他大概率在哪个范围内——正态分布。

对量化交易的意义

  • 不需要预测股价明天涨还是跌
  • 只需要知道波动范围的概率分布
  • 在概率有利的时候下注，长期就能盈利

这就是为什么量化交易的核心是管理概率，而不是预测价格。

正态分布

  • 正态分布核心概念

    • 钟形曲线，中间概率高，两端概率低，左右对称
    • μ（均值）决定中心位置，σ（标准差）决定胖瘦
    • 曲线下面积总和 = 1（概率总和100%）
    • μ=0, σ=1 是标准正态分布

  • 对股价的意义

    • 股价每天的涨跌幅近似服从正态分布，大涨大跌概率小，小幅波动概率大。所以可以用正态分布来模拟股价走势

  • 生成符合正态分布的随机数组，是量化中模拟股价数据的常用方法。

loc=0, scale=1时为标准正态分布。

概率与市场博弈

  • 股票涨跌 = 随机事件，不能确定结果，但可以用概率量化其可能性。
  • 量化交易的本质就是找到概率上有优势的策略，而不是预测具体涨跌。

例子：动量策略
连续上涨N天 → 买入，连续下跌N天 → 卖出。
策略背后的逻辑不是”预测”，而是历史统计上这种”惯性”现象胜率大于50%。

伯努利分布

伯努利分布 = 二项分布，只有两种结果（0或1）的概率分布。

  • 单次试验 = 伯努利试验
  • n次独立重复试验 = 二项分布，记作 X～B(n, p)

n次试验中某事件恰好发生k次的概率

  • C(n,k) = 组合数，k次发生的所有可能排列方式
  • p^k = k次成功的概率
  • (1-p)^(n-k) = 剩余n-k次失败的概率

抛5次硬币全部正面：C(5,5) * 0.5^5 * 0.5^0 = 1/32 = 3.125%

关键认知：独立性

  • 每次试验相互独立，前5次结果对第6次没有任何影响。
  • 32种组合（00000到11111）每种概率都是1/32，连续5次正面和10011出现的概率完全一样。

对量化交易的启示

  • 短期胜率高不代表策略真的好。5次交易3胜2负看似60%胜率，但随着交易次数增加，胜率会趋向50%。真正的策略优势需要大量样本验证，而不是短期表现。
  • 赌场不怕你赢，因为独立试验的长期概率必然回归均值。量化的目标是找到真正能把胜率推高到50%以上的策略，而不是靠短期运气。

三种市场模型对比

模型1：50%胜率，无手续费（零和游戏）

逻辑：亏一次下次加注3倍，理论上总会赢回来。

主力资金流入流出

为什么要筛选主力大单

  • 散户资金量小，买卖对股价影响微乎其微。主力（机构、大户）一笔大单就能推动股价明显波动，所以追踪大单 = 追踪真正能影响股价的资金动向，参考价值更高。

成交额 = 价格 × 成交量，按买盘/卖盘/中性盘分类统计。

资金定义（东方财富标准）

  • 超大单：≥50万股或≥100万元
  • 大单：≥10万股或≥20万元
  • 主力流入 = 超大单+大单买入成交额
  • 主力流出 = 超大单+大单卖出成交额
  • 净额 = 流入 - 流出
  • 净比 = (流入-流出) / 总成交额

除权数据前复权和后复权处理

除权除息

公司分红或送股后，股价会出现下跌缺口，叫除权缺口。这个跳空不是真实的下跌，直接用除权价格计算涨跌幅会严重失真。

例子：股价20元，每10股送10股，除权后变10元，股本翻倍。表面上跌了50%，实际价值没变。

前复权

保持现有股价不变，调整以前的历史价格。最近的价格是真实价格，往前的价格等比缩小。

后复权

保持最早的股价不变，调整除权后的价格。从起点开始，往后的价格等比放大。

量化必须用复权数据

用除权数据做回测，计算出来的涨跌幅是错的，策略结果完全失真，没有参考价值。

复权涨跌幅

真实涨跌幅 = 复权后的涨跌幅，是前后复权计算的基础。

例子：股价17.64元，每10股转增10股分红5.5元：

前复权 vs 后复权选哪个

回测策略用前复权，因为最近的价格是真实价格，历史价格向前调整，不影响当前判断。后复权起点不同结果也不同，更适合长期持有收益计算。

成交量

成交量原理

  • 成交量反映市场参与度，价格变动需要成交量配合才有意义
  • 量价齐升 — 上涨有支撑，主力在吸筹或拉升
  • 价涨量缩 — 上涨动力不足，需警惕
  • 天量出货 — 股价冲高但成交量异常放大，主力可能在出货

单看K线容易被假突破迷惑，结合成交量可以过滤掉很多噪音信号。

MACD

MACD 称为异同移动平均线，属于趋势类指标，由快速线 DIF、慢速线 DEA 组成。DIF 和 DEA 以零轴为中心上下交叉移动，红绿柱状图 BAR 为辅助显示指标。计算方法如下

  • DIF（差离值）= EMA12 - EMA26
  • DEA（差离平均值）= DIF 的 9 日移动平均线
  • BAR（柱状值）= 2 × (DIF - DEA)

在持续上涨中，短期 EMA 在长期 EMA 之上，正离差值会越来越大，BAR 柱也会扩展。当行情即将反转时，DIF 与 DEA 距离会逐渐缩小，出现背离，预示趋势强弩之末。

KDJ

KDJ 是最常用的震荡类指标，用于衡量市场超买或超卖现象。计算方法如下

  • RSV = (收盘价 - 最近 n 日最低价) ÷ (最近 n 日最高价 - 最近 n 日最低价) × 100
  • K 值 = 2/3 × 前一日 K 值 + 1/3 × 当日 RSV
  • D 值 = 2/3 × 前一日 D 值 + 1/3 × 当日 K 值
  • J 值 = 3 × K 值 - 2 × D 值

K、D 值范围在 0～100 之间。K 值上穿 D 值为金叉（买入信号），下穿为死叉（卖出信号）。J 值超过 100 为超买区，低于 0 为超卖区。

K 线

K线（Candlestick）起源于18世纪日本米市交易，记录每个交易日四个价格：开盘价、收盘价、最高价、最低价

  • 为什么要看K线 ？

单看收盘价只知道今天涨了还是跌了，但K线能告诉你更多

  • 实体大小 — 实体越大说明多空分歧越大，当天价格波动剧烈
  • 影线长短 — 长下影线说明盘中被打压但最终收回，有买盘承接；长上影线说明冲高后被砸回，有卖盘压制
  • 形态组合 — 连续几根K线组合在一起可以判断趋势的延续或反转，比如”早晨之星”预示底部反转，”乌云盖顶”预示顶部反转

一个直观的例子

同样是今天收盘价比昨天涨了1%，但

  • 如果是一根小实体加长上影线，说明盘中冲得很高但被压回来，上方卖压很重
  • 如果是一根大红实体，说明全天强势上涨，多头主导

K线本身是一个时间周期的概念，一根K线代表一个周期内的开高低收，这个周期可以是

  • 日K线 — 一根K线 = 一天，最常用
  • 周K线 — 一根K线 = 一周
  • 月K线 — 一根K线 = 一个月
  • 分钟K线 — 一根K线 = 1分钟/5分钟/15分钟，短线交易者用

移动平均线 (SMA)

移动平均线（MA）是把过去N天的收盘价加起来除以N，每天滚动更新。

  • Ma20 — 过去20天的平均收盘价
  • Ma30 — 过去30天的平均收盘价
  • Ma60 — 过去60天的平均收盘价

Ma20上每一个点代表什么？

  • 比如今天是12月20日，那今天Ma20这个点 = 12月1日到12月20日收盘价的平均值。明天这个点 = 12月2日到12月21日的平均值。每天往前滚动一天，所以叫”移动”平均线。

为什么要看均线？

  • 单看K线只能看到今天涨跌，但看不出整体趋势。均线把短期波动平滑掉，让你看清楚市场的大方向。

均线代表什么？

  • 均线本质上是持股人的平均成本线。比如Ma20是过去20天买入的人的平均买入价格
    • 股价在均线上方 — 大部分人在盈利，市场情绪偏多
    • 股价在均线下方 — 大部分人在亏损，市场情绪偏空

三条均线怎么配合看？

  • Ma20 反应最灵敏，短期趋势
  • Ma30 中期趋势
  • Ma60 反应最迟钝，长期趋势

三条均线从上到下排列（Ma20 > Ma30 > Ma60）叫多头排列，说明短中长期趋势都向上，是强势市场的标志。反过来叫空头排列，趋势向下

成交量

  • 成交量是当天交易日内成交的股票数量，以手为单位（1手 = 100股）。

为什么要看成交量？

  • K线只告诉你价格涨跌，但不告诉你这个涨跌有多少人参与。成交量就是参与度的体现——同样是涨1%，100个人买和10000个人买，意义完全不同。

成交量怎么看？

  • 量价齐升 — 价格涨，成交量也放大，说明大量资金涌入，上涨有支撑
  • 价涨量缩 — 价格涨但成交量萎缩，说明参与的人越来越少，上涨动力不足，需要警惕
  • 天量出货 — 股价冲高但成交量异常放大，往往是主力在高位卖出，散户在接盘

成交量柱子颜色

  • 红色柱子 — 当天收盘价高于开盘价，当天上涨
  • 绿色柱子 — 当天收盘价低于开盘价，当天下跌
  • 柱子越高说明当天成交量越大

单看K线容易被假突破迷惑，结合成交量可以过滤掉很多噪音信号。主力拉高出货时，K线看起来是上涨，但成交量异常放大就是卖出的信号。

这张图完美印证了”天量出货”的概念——成交量异常放大但股价不涨反跌，就是主力在出货的信号。散户看到大涨追进去，结果成了主力的接盘侠。

MACD

MACD（Moving Average Convergence Divergence）异同移动平均线，属于趋势类指标。

MACD 由三部分组成

  • DIF（快线）= 12日EMA - 26日EMA
  • DEA（慢线）= DIF 的9日移动平均线
  • BAR（柱状图）= 2 × (DIF - DEA)，正值红色，负值绿色

EMA 是什么？

  • EMA（指数移动平均线）和普通均线类似，区别是越近的价格权重越大，对最新价格反应更灵敏。

EMA 和 SMA 的区别

  • SMA — 每天权重相同，昨天和10天前的价格对结果影响一样大
  • EMA — 越近的价格权重越大，对最新价格反应更灵敏，均线拐点出现得更早

EMA 是从收盘价计算出来的，公式

EMA今天 = 前一天EMA × (1 - 平滑系数) + 今天收盘价 × 平滑系数

  • 平滑系数 = 2 ÷ (N + 1)，N是周期
  • EMA12 的平滑系数 = 2 ÷ 13 ≈ 0.15
  • EMA26 的平滑系数 = 2 ÷ 27 ≈ 0.07

怎么看 MACD？

  • DIF 上穿 DEA → 金叉，买入信号
  • DIF 下穿 DEA → 死叉，卖出信号
  • BAR 柱由绿转红 → 趋势由跌转涨
  • BAR 柱由红转绿 → 趋势由涨转跌

零轴的意义

  • DIF 在零轴上方 → 短期均线高于长期均线，整体趋势向上
  • DIF 在零轴下方 → 短期均线低于长期均线，整体趋势向下

MACD 本质上反映的是短期和长期趋势之间的距离。距离越拉越大说明趋势在加速，距离开始缩小说明趋势在减弱，预示可能反转。

KDJ

KDJ 是最常用的震荡类指标，用来判断市场是否超买或超卖。

为什么要看 KDJ？

  • 均线和 MACD 告诉你趋势方向，但不告诉你现在价格是贵了还是便宜了。KDJ 就是用来衡量当前价格在最近一段时间高低区间里的相对位置。

KDJ 怎么计算？

  • 核心思想：今天收盘价在最近9天的高低区间里处于什么位置
  • 收盘价靠近最高价 → RSV 值高 → 市场偏贵，可能超买
  • 收盘价靠近最低价 → RSV 值低 → 市场偏便宜，可能超卖

RSV（Raw Stochastic Value）未成熟随机值，是计算 KDJ 的第一步

RSV = (今日收盘价 - 最近N日最低价) ÷ (最近N日最高价 - 最近N日最低价) × 100

直白说就是：今天收盘价在最近9天价格区间里处于什么位置，用百分比表示。

  • RSV = 100 → 今天收盘价就是最近9天最高价
  • RSV = 0 → 今天收盘价就是最近9天最低价
  • RSV = 50 → 今天收盘价在最近9天的中间位置

由 RSV 推导出 K、D、J 三条线

  • K 值 — 对 RSV 做平滑，反应比较灵敏
  • D 值 — 对 K 值再做平滑，反应比较迟钝
  • J 值 — 3K - 2D，放大 K 和 D 的差距，反应最灵敏

怎么看 KDJ？

  • K 上穿 D → 金叉，买入信号
  • K 下穿 D → 死叉，卖出信号
  • KDJ 都在 80 以上 → 超买区，注意风险
  • KDJ 都在 20 以下 → 超卖区，可能反弹

KDJ 在震荡行情里很好用，但在单边趋势行情里容易钝化——强势上涨时 KDJ 会长期停在超买区，一直发出卖出信号，但股价还在涨。所以要结合 MACD 和均线一起看。
