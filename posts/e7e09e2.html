量化学习

发表于2025-06-13|更新于2026-04-23

什么是量化

  • 量化交易 = 用数据+统计+程序，系统化地找出历史上大概率盈利的规律，然后自动执行。核心是去除人为情绪，靠模型和算法驱动决策。起源于美国，国内近年快速兴起。

什么是量化交易

  • 核心问题：市场 = 负和游戏（手续费是house edge），随机交易必亏。
  • 解法：找到 win_rate > 50% 的信号，只在有优势时入场。
  • 量化交易本质：从历史数据里挖出 P(win) > 0.5 的规律，系统化执行，去掉人为情绪干扰。
  • 核心思想：模型会过期，市场会变，策略必须持续迭代。
  • 三个关键模块：
    • 策略回测 → 验证历史胜率
    • 仓位管理 → 控制每次下注大小
    • 止盈止损 → 保护期望收益

类比：不是赌博，是在找一个有正期望值的游戏，然后反复执行它。

量化交易基本流程

数据采集 → 数据清洗 → 量化模型 → 交易信号 → 执行下单

  • 三个核心环节

    • 数据层 — 历史行情 + 基本面 + 新闻资讯，清洗后喂给模型
    • 模型层 — 数学建模 + 策略编程，分析数据产生信号
    • 执行层 — 信号转化为具体指令：买什么、何时买、买多少、何时卖

  • 对程序员来说本质就是一个 pipeline：数据进，信号出，自动执行。

回测的四大陷阱

  • 交易者效应 — 你的”漏洞”被其他人发现后就被修补了。
  • 随机效应 — 过拟合了噪声，不是真规律。
  • 最优化矛盾 — 参数调出来的是历史最优，不是未来最优。
  • 过度拟合 — 训练集准确率100%，测试集崩了。

结论： 回测只是参考，不是答案。量化的真正壁垒是持续学习、持续更新模型的能力，而不是找到一个”永动机策略”。

量化交易发展历程

  • 索普（Edward Thorp） — 开创者。数学家出身，先用概率+凯利公式横扫拉斯维加斯，1967年出版量化交易开山之作，1969年创立史上第一家量化对冲基金。本质是把赌场的数学优势逻辑搬到了华尔街。

  • 西蒙斯（James Simons） — 集大成者。1988年创立文艺复兴科技，旗舰大奖章基金1989-2009年均回报约35%，远超巴菲特、索罗斯，2008年金融危机照样稳。把量化交易推向了这个时代的主流。

  • 索普开创 → 量化产品涌现（指数基金、对冲基金、FOF）→ 西蒙斯验证并推广 → 成为全球资管主流

核心提醒： 量化不是万能的，西蒙斯自己的基金也有亏损年份。优势在于长期概率，不在于每次必赢。

随机漫步理论

  • 核心思想：不可预测 ≠ 不可量化
  • 想象一个醉汉站在街道中间，每走一步完全随机：往前或往后概率各50%。

  • 对应股价：

    • 每天涨跌 = 醉汉每步前进后退
    • 今天股价 = 醉汉当前位置

  • 随机漫步理论说股价每步随机，无法预测具体走向。但索普发现，虽然不知道醉汉走到哪，但知道他大概率在哪个范围内——正态分布。

  • 对量化交易的意义：

    • 不需要预测股价明天涨还是跌
    • 只需要知道波动范围的概率分布
    • 在概率有利的时候下注，长期就能盈利

这就是为什么量化交易的核心是管理概率，而不是预测价格。

正态分布

  • 正态分布核心概念：

    • 钟形曲线，中间概率高，两端概率低，左右对称
    • μ（均值）决定中心位置，σ（标准差）决定胖瘦
    • 曲线下面积总和 = 1（概率总和100%）
    • μ=0, σ=1 是标准正态分布

  • 对股价的意义：

    • 股价每天的涨跌幅近似服从正态分布，大涨大跌概率小，小幅波动概率大。所以可以用正态分布来模拟股价走势

  • 生成符合正态分布的随机数组，是量化中模拟股价数据的常用方法。
  • loc=0, scale=1时为标准正态分布。

numpy矢量运算

  • 矢量运算的意思是：对数组的操作会自动作用到每一个元素，不需要for循环。
  • 涨跌幅公式：(今日收盘价 - 昨日收盘价) / 昨日收盘价

  • 返回[low, high)之间的随机整数，包含low，不包含high
  • high不填时，生成[0, low)之间的随机整数

随机漫步轨迹核心逻辑

  • np.where(condition, x, y) — 满足条件取x，否则取y，矢量运算版的三元表达式。
  • cumsum() — 累积求和，[1,-1,1,1] → [1,0,1,2]，用来追踪位置。

1000次随机漫步模拟

  • alpha=0.05 — 透明度极低，1000条线叠加后颜色深浅反映概率密度，越深说明醉汉出现在该位置的概率越大。

单次漫步位置不可预测，但1000次模拟后可以看出：随着步数增加，可能出现的位置范围扩大，但距离起点越远概率越小，符合正态分布。这就是用统计学量化”不可预知”的核心思想。

概率与市场博弈

  • 股票涨跌 = 随机事件，不能确定结果，但可以用概率量化其可能性。
  • 量化交易的本质就是找到概率上有优势的策略，而不是预测具体涨跌。

例子：动量策略
连续上涨N天 → 买入，连续下跌N天 → 卖出。
策略背后的逻辑不是”预测”，而是历史统计上这种”惯性”现象胜率大于50%。

伯努利分布

  • 伯努利分布 = 二项分布，只有两种结果（0或1）的概率分布。
    • 单次试验 = 伯努利试验
    • n次独立重复试验 = 二项分布，记作 X～B(n, p)

n次试验中某事件恰好发生k次的概率

P(X=k) = C(n,k) * p^k * (1-p)^(n-k)

  • C(n,k) = 组合数，k次发生的所有可能排列方式
  • p^k = k次成功的概率
  • (1-p)^(n-k) = 剩余n-k次失败的概率

抛5次硬币全部正面：C(5,5) * 0.5^5 * 0.5^0 = 1/32 = 3.125%

  • 关键认知：独立性

    • 每次试验相互独立，前5次结果对第6次没有任何影响。
    • 32种组合（00000到11111）每种概率都是1/32，连续5次正面和10011出现的概率完全一样。

  • 对量化交易的启示

    • 短期胜率高不代表策略真的好。5次交易3胜2负看似60%胜率，但随着交易次数增加，胜率会趋向50%。真正的策略优势需要大量样本验证，而不是短期表现。
    • 赌场不怕你赢，因为独立试验的长期概率必然回归均值。量化的目标是找到真正能把胜率推高到50%以上的策略，而不是靠短期运气。

概率在市场中的应用

  • 三种市场模型对比

模型1：50%胜率，无手续费（零和游戏）


结果：赚钱和亏钱各一半，符合零和游戏特征。

模型2：50%胜率，有手续费（负和游戏）


结果：几乎所有人血本无归。手续费看似很小，但随交易次数增加累积效应显现，零和变负和。

模型3：越亏越加注（一条路走到黑）


逻辑：亏一次下次加注3倍，理论上总会赢回来。
问题：需要无限本金，遇到黑天鹅连续亏损10+次，散户根本撑不住。

模型4：仓位管理（逆袭成功）


逻辑：胜率高时加大仓位，胜率低时减小仓位，赢多亏少。
结果：50人全部盈利。

iterrows()生成器（次慢）

  • iterrows()是对DataFrame行迭代的生成器，每次返回行索引和行数据，比for..in快但仍是逐行处理。

  • 利用类Cython迭代器优化，比iterrows()快很多。

  • Pandas series矢量化
